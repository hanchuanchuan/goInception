

### 支持参数


goInception的审核规则可以通过```inception show variables;```查看

```sql
inception show variables;
```

支持以下方式设置:

- 1.通过```inception set ```设置

```sql
inception set check_dml_limit = true;
```

- 2.配置config.toml,并通过```-config=config.toml```指定配置文件启动
